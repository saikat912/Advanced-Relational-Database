# Advanced Relational Database and SQL

A hands-on project focused on advanced SQL concepts, including functions, stored procedures, triggers, and recursive queries. This project demonstrates how to build efficient, maintainable, and scalable database applications using advanced relational database techniques.

---

## 📖 Course Overview

This course focuses on mastering advanced SQL concepts and applying them to solve real-world database problems. It also covers techniques for generating end-user-friendly data presentations using SQL.

### Learning Objectives

* Understand and implement advanced SQL concepts.
* Create and use SQL functions.
* Develop stored procedures for reusable database logic.
* Automate database operations with triggers.
* Write Common Table Expressions (CTEs).
* Implement recursive CTEs for hierarchical data.
* Generate meaningful reports and data presentations using SQL.

By the end of this project, you will be able to write complex SQL queries and develop advanced database applications.

---

## 📂 Project Structure

The project consists of the following tasks:

### Task 1: User Interface Orientation

* Explore the virtual workspace.
* Understand the project environment.

### Task 2: Server Functions

* Learn SQL server functions.
* Write and execute custom server functions.

### Task 3: Stored Functions

* Create reusable stored functions.
* Return calculated values from SQL functions.

**Optional Assessment**

* Practice exercises after Task 3.

### Task 4: Stored Procedures

* Create stored procedures.
* Pass parameters.
* Execute procedures efficiently.

### Task 5: Triggers

* Create database triggers.
* Automate actions after INSERT, UPDATE, and DELETE operations.
* Call stored procedures using triggers.

**Optional Practice**

* Time for a Sale! Challenge

### Task 6: Common Table Expressions (CTEs)

* Create readable SQL queries.
* Simplify complex joins and aggregations.

### Task 7: Recursive Common Table Expressions

* Query hierarchical data.
* Build recursive queries for parent-child relationships.

**Optional Final Challenge**

* Supply and Demand Challenge
